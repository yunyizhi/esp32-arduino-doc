Arduino 数字IO

digitalRead()

用于读取高低电平。

int digitalRead(uint8_t pin);

  • pin: 要读取的引脚编号。

返回

HIGH或LOW。

注意事项和警告

悬空引脚读取可能是高或者低。

digitalWrite()

用于输出高低电平 ，例如点亮板载LED灯。

void digitalWrite(uint8_t pin, uint8_t val)

  • pin: 要输出的的引脚编号,HIGH或LOW。

  • val: HIGH或LOW。
